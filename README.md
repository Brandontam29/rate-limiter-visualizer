## How to run the project

### Frontend

Install dependencies

`cd frontend`

`npm install`

Run React website

`npm run dev`


### JS Backend

Open new terminal and install dependencies

`cd js-backend`

Install dependencies

`npm install`

Run Express server

`npm run dev`
